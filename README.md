# Secure ERC-20 Token (Anti-Whale & Tax Logic)

Professional smart contract implementation for Ethereum/EVM networks.

### Features:
- **Anti-Whale Mechanism:** Limits the maximum transaction amount to protect the market from sudden dumps.
- **Automated Tax System:** 2% fee on every transaction automatically sent to the owner wallet.
- **Security First:** Built using OpenZeppelin standards.

### Use Cases:
Ideal for community-driven projects, meme coins, or any ecosystem requiring stable growth and developer funding.
